Xoxo#

This is a small unstructured grid consisting of two unconnected parts. This is downloaded via xugrid.data.xoxo(). The topology data is downloaded to a local directory if itâ€™s not there already.

import matplotlib.pyplot as plt

import xugrid

grid = xugrid.data.xoxo()

fig, ax = plt.subplots()
xugrid.plot.line(grid, ax=ax, color="#bd0d1f")
ax.set_xlim([0.0, 100.0])
ax.set_ylim([0.0, 85.0])
ax.set_aspect(1)
