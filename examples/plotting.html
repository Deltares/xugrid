Plot unstructured mesh dataÂ¶

Xarray provides a convenient way of plotting your data, provided it is structured. xugrid contains a few additional plotting functions to easily make spatial plots of unstructured grids.

Out:

<matplotlib.collections.LineCollection object at 0x7f0ec89c2580>


import matplotlib.pyplot as plt

import xugrid

ds = xugrid.data.disk()
uda = ds["face_z"]

uda.ugrid.plot()
uda.ugrid.plot.face()
uda.ugrid.plot.face.pcolormesh()
uda.ugrid.plot.face.contour()
uda.ugrid.plot.face.contourf()
uda.ugrid.plot.face.imshow()


fig, ax = plt.subplots()
uda.ugrid.plot(ax=ax)

uda.ugrid.plot.edge(color="black", linewidth=1.0)
